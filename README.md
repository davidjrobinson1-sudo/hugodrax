# RINGO Website

Static GitHub Pages-ready website for RINGO NFC payment rings.

## Upload to GitHub

1. Upload all files in this folder to a GitHub repository.
2. In GitHub, open **Settings > Pages**.
3. Set the source to the repository branch that contains `index.html`.
4. Save and wait for GitHub Pages to publish.

## Files

- `index.html` - page content
- `styles.css` - layout and visual design
- `script.js` - mobile menu and reservation selection
- `assets/` - logo, product renders, packaging, and lifestyle image

Before launch, replace `hello@ringo.example` in `index.html` and `script.js` with your real reservation/contact email.
